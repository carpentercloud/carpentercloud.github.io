### 复制一个文件
```
cp file1 file2
```
### 移动文件
```
mv fileName tagetFile
```
### 移动文件夹

### 新建文件
```
touch fileName
```
### 新建文件夹（即目录）
```
mkdir name
```
### 删除文件
```
rm fileName
```
### 删除文件夹
```
rm -rf foldername
```

### 找自己电脑上的公钥
```
cd ~/.ssh
```
### 查看端口占用进程
```
ps aux | grep 6375
```

### 杀死进程
```
kill 6375
```

### 具体如何解读
```
ssh -N -f -L 6375:127.0.0.1:9999 admin@123.207.242.200
```
